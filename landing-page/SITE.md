# CM Hair Salon — Site Vision & Roadmap
**Stitch Project ID:** `17736084193682925594`

## 1. Vision
A luxury single-page website prototype for CM Hair Salon (5 branches) that communicates premium quality, modern aesthetics, and accessibility. The design language "Warm Noir" uses warm copper and espresso tones to differentiate from typical gold-based luxury.

## 2. Technology
- React + Vite + TypeScript
- Framer Motion (animations)
- EmailJS (contact form)
- Dark/Light mode toggle

## 3. Target Audience
Indian clientele seeking premium hair salon services across 5 branches. Walk-ins and appointment-based.
